How to install in Outlook

Outlook on Windows (new + classic)

  1. Click Copy signature above.
  2. In Outlook, go to File › Options › Mail › Signatures (classic) or Settings › Accounts › Signatures (new Outlook).
  3. Click New, give it a name (e.g. "Intercare"), and paste with Ctrl + V into the editor.
  4. Set it as the default for New messages and Replies/forwards, then Save.

Outlook on Mac

  1. Click Copy signature.
  2. Open Outlook › Settings › Signatures.
  3. Click +, paste with ⌘ + V, name it, close to save.
  4. Under Choose default signature, pick your new signature.

Outlook on the Web

  1. Click Copy signature.
  2. In Outlook Web, click the gear › Mail › Compose and reply.
  3. Click + New signature, paste in, save, and select it as default.

Outlook mobile app (iPhone / Android)

  1. Open this page on your phone and tap Copy signature (or tap Download .htm file, open it, then long-press › Select All › Copy).
  2. Open the Outlook app and tap your account icon (top left).
  3. Tap the gear icon › Signature.
  4. If you have multiple accounts, turn on Per Account Signature and pick your Intercare account.
  5. Tap into the signature field, clear the default text, then long-press › Paste.
  6. Tap the checkmark (iOS) or back arrow (Android) to save.

Note: the Outlook mobile app strips most formatting from signatures — expect a plain-text version of your name, title, phone, and email rather than the full branded card. For the full signature, install it on desktop or web Outlook (above); it will still appear correctly on emails sent from any device once saved server-side.

If "Copy signature" isn't working

Some browsers block rich-clipboard. Click Download .htm file, open the file in a browser, press Ctrl/⌘ + A to select all, then Ctrl/⌘ + C to copy, and paste into Outlook.
